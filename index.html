Lesson 6: CSS

Как поключать CSS

  1. inline стили - <h1 style="color:blue;margin-left:30px;">This is a heading</h1>
  2. стили в том же файле - тэг <style>тут описание стилей</style>
  3. внешний файл - <link rel="stylesheet" type="text/css" href="mystyle.css">
